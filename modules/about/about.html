Pop quiz // Which JavaScript framework did I use to build my website

A) React

B) Vue

C) Angular

✅ It’s written from scratch with pure HTML, JS and CSS.

The trendy thing to do is use a single page application framework like the ones mentioned above and maybe also a CSS library. But writing from scratch makes things unique as every bit is handmade with a touch of the author’s flair. Pragmatically though, writing code from scratch makes implementing niche features so much easier because you won’t have to conform to the rules of the framework you are using. And I expect to be writing a lot of niche stuff on this website like music apps.

Oh, and the entire build system for this site was handwritten too.
